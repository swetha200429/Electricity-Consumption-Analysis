# Electricity-Consumption-Analysis
📊 India’s Electricity Consumption Analysis using NumPy

🔹 Project Description

This project focuses on analyzing India’s electricity consumption data using NumPy.
The goal is to extract meaningful insights from a real-world dataset by performing numerical computations and trend analysis.

The project demonstrates how NumPy can be used for efficient data processing and analysis in real-life scenarios.

🔹 Objectives

Analyze electricity consumption trends over time
Identify peak and low consumption periods
Perform state-wise comparison
Apply NumPy operations for efficient computation
Visualize data for better understanding

🔹 Dataset Information

Dataset: India’s Electricity Consumption Dataset
Contains:
Dates
State-wise electricity consumption
Total consumption

🔹 Technologies Used

Python 🐍
NumPy 📊
Pandas 📁
Matplotlib 📈

🔹 Key Features

Conversion of dataset into NumPy arrays
Statistical analysis (mean, sum, standard deviation)
Peak consumption day detection
Daily consumption trend analysis
State-wise average consumption comparison
Monthly analysis using grouped data
Data visualization using line plots, histograms, and bar charts

🔹 NumPy Concepts Used

Array operations
Aggregation functions (sum, mean, max, min)
argmax, argmin
diff() for trend analysis
Multi-dimensional arrays
Axis-based operations
